router.on

router.on(path, config)

添加一条顶级的路由配置。在内部实现时，router.map() 对于接收到的路由映射对象中每个键值对都调用 router.on() 。

参数

  • path: String - 查看路由匹配
  • config: Object - 查看路由配置对象.

例子

router.on('/user/:userId', {
  component: {
    template: '<div>{{$route.params.userId}}</div>'
  }
})
