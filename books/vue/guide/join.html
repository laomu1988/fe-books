加入 Vue 社区！

Vue.js 的社区在持续增长中，欢迎为社区提供一臂之力。事实上，一个增长的社区是这个开源项目是否健康、是否值得投入的最重要的一个标志。

如果你也想成为社区的一员，此页正是为你提供帮助的。看看下文了解 Vue.js 这艘船的航向等等。

欢迎上船！

你应知道的！

社区

  • 论坛 - 提问答疑的最好地方。
  • Gitter - 开发者聊天室。在这里你可以提问，不过最好在论坛提问，因为论坛有版块。
  • Github - 报告问题，提交请求，为你所做的任何贡献表示深深感谢！

第三方资源

  • Awesome Vue - 看看其他牛人发布的好资源。
  • "Show and Tell" 子论坛 - 另一个好地方，看看他人的作品。

参与 Vue 开发

  • 参与规则 - 和其它社区一样，我们有规矩，这是为了让合适的人做合适的事。

  • Vue 的主要组件

    • Vue - 核心库。
    • Vuex - 类 Flux 的应用架构。
    • Vue-router - 单页面应用路由。
    • Vue-resource - 网络请求插件，使用 XMLHttpRequests 或 JSONP 处理响应。
